# AGENTS.md — AI開発者(Codex/Gemini/Cursor等)への指示

## プロジェクトの目的
4人家族の毎日の「在庫→献立→買い物→レシート→在庫」ループを回すWebアプリ。
**スマホでの使いやすさ > 機能の多さ > 技術的な美しさ** の優先順位を絶対に守ること。

## 現在の構成
- `index.html` 1ファイル完結(HTML+CSS+Vanilla JS)。ビルド不要。
- 永続化は localStorage キー `kondate-loop-v1`。スキーマは `docs/DATA_MODEL.md`。

## 変更時の絶対ルール
1. **既存ユーザーデータを壊さない。** スキーマ変更時は `load()` にマイグレーション処理を追加し、
   `DATA_MODEL.md` のバージョン履歴を更新すること。
2. **1タスク=1機能で小さく変更する。** `docs/TODO.md` から1項目選び、完了したらチェックを付ける。
3. `esc()` を通さずユーザー入力をinnerHTMLに入れない(XSS)。
4. 動作確認は `docs/TEST_PLAN.md` の該当項目を実施。
5. Next.js移行は ROADMAP Phase 2 の判断基準を満たすまで行わない。

## コード規約
- 日本語UIテキスト。関数は機能ごとにセクションコメント(`/* ==== 在庫 ==== */`)配下に置く。
- 新しい依存ライブラリはCDN経由で1つまで。原則ゼロ依存を維持。
- CSSカスタムプロパティ(`:root`)の色トークンのみ使用。ハードコード色禁止。

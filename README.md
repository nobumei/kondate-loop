# こんだてループ 🍳

在庫 → 献立 → 買い物 → レシート → 在庫。家庭の食事管理ループを毎日回すWebアプリ。

## 使い方(今すぐ)

1. `index.html` をスマホ/PCのブラウザで開くだけ。ビルド・サーバー不要。
2. スマホなら「ホーム画面に追加」でアプリのように使える。
3. データは端末のlocalStorageに保存。機種変更前に「設定 > エクスポート」。

### 公開したい場合(5分)
- GitHub Pages: リポジトリにpushして Settings > Pages を有効化
- Netlify Drop: https://app.netlify.com/drop にフォルダをドラッグ

## 家族で同期する

1. 端末A: 設定タブ > 家族で同期 > 「あいことばを生成」>「同期を開始」
2. あいことばを家族にLINE等で共有
3. 家族の端末: 同じあいことばを入力して「同期を開始」

注意: あいことばを知っている人は誰でも読み書きできる。推測されにくい文字列(生成ボタン推奨)を使うこと。
同時に別端末で編集すると、後から保存した方が勝つ(直近の変更が消えることが稀にある)。
